## üöÄ How to Execute

Clone the project and access the folder.

```bash
$ git clone https://github.com/FilipiMachado/e-commerce-website-vuejs.git
$ cd e-commerce-website-vuejs
```

To start, follow this steps:
```bash
# Install the dependencies
$ npm install

# Start the project
$ npm run serve
```
